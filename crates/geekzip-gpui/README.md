# GeekZip GPUI

GeekZip 的 Rust 原生桌面界面，使用 GPUI 与 gpui-component 构建，并直接依赖
`geekzip-core`。

## 运行

```bash
cargo run -p geekzip-gpui
```

首次构建需要编译 GPUI/Metal 图形依赖，后续增量构建会明显加快。

## 当前迁移状态

- 已完成：原生窗口、点阵黑绿主题、Normal/Pro 模式、核心导航、真实空状态、
  文件详情、日志与资源状态栏。
- 已连接：系统文件选择器、Rust 后台任务、单文件解压、ZIP 压缩、递归解压、
  文件夹批量解压、目录自动解压。
- 密码本保存在 `~/.geekzip/passwords.json`。加密 ZIP 会自动尝试密码本候选，
  成功密码也会自动写回。
- 操作页会显示当前状态、结果与错误日志。更细粒度的字节级实时进度仍待核心层
  增加进度回调后接入。
- 首页已移除演示任务、演示归档、伪文件详情与伪日志，只呈现用户实际选择和操作
  产生的数据。
- 底部状态栏和任务面板会实时显示系统 CPU、GeekZip 进程 CPU、GPU、内存和线程数。
  当前归档解压引擎使用 CPU 与磁盘 I/O；GPU 数据用于观察整机图形负载，不表示
  解压任务正在使用 GPU。

桌面应用入口为 `geekzip-gpui`。
